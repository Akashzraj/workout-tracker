# Workout Tracker

A basic web app to track workouts. Built with Node.js, Express, MongoDB, and HBS.

## Live Site

https://workout-tracker-sdf2.onrender.com

## Features

- Register and log in
- View workouts
- Add, edit, and delete workouts
- GitHub login

## Tech Used

- Node.js
- Express
- MongoDB (Mongoose)
- Handlebars (HBS)
- Passport.js
- Bootstrap

## Extra Feature

- GitHub login using Passport
